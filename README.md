# Ai x Lab Studio Clone

This workspace contains a local copy of the Ai x Lab portfolio template editor.

## Run locally

```powershell
python -m http.server 4173
```

Open `http://127.0.0.1:4173/` in a browser.

The app is intentionally kept as a static HTML/CSS/JavaScript implementation so its editor state, template animations, media crop flow, JSON import/export, HTML/JPG export, and publish preview remain available without a build step.
